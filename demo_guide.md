# 🏥 종합검진 AI 솔루션 기능 및 시연 가이드

이 문서는 영업팀이 솔루션의 **기능과 작동 원리**를 정확히 이해하고 시연할 수 있도록 작성된 가이드입니다. 각 화면이 어떤 역할을 하는지, 내부적으로 어떤 로직이 돌아가는지 설명합니다.

---

## 1. 화면별 기능 상세 설명

### 1.1 📄 종합소견 생성 (Generation / Dashboard)
로그인 후 처음 만나는 메인 대시보드 화면입니다. **새벽 자동 스케줄링(배치)의 결과를 확인하고 관리**하는 곳입니다.

*   **핵심 로직: 4단계 자동 분석 시스템 (새벽 배치)**
    이 화면의 **[배치 자동 생성]** 버튼은 실제 서버에서 **매일 새벽 6시**에 자동으로 돌아가는 로직을 수동으로 실행하는 기능입니다.
    1.  **준비 (Initialization)**: 검진 후 5일이 지난 환자 데이터를 수집합니다.
    2.  **수치 분석 (Numeric)**: 혈액, 소변 등 수치형 데이터를 그룹별로 1차 정밀 분석합니다. (임상 가이드라인 참고)
    3.  **텍스트 분석 (Text)**: 내시경, 초음파 등 판독문(글자)을 그룹별로 2차 정밀 분석합니다.
    4.  **종합 판단 (Comprehensive)**: 1, 2단계의 분석 결과를 모두 모아, AI가 전체적인 건강 상태를 요약하고 위험도를 판단하여 최종 소견을 완성합니다.

*   **주요 기능**:
    *   **검진 대상자 조회**: 날짜별로 검진이 완료된 환자 리스트를 확인합니다.
    *   **소견 생성 상태 확인**: AI 분석이 완료되었는지(Completed), 대기 중인지(Pending) 한눈에 파악할 수 있습니다.
    *   **결과지 바로가기**: 생성 완료된 환자의 눈 아이콘(👁️)을 클릭하면 결과 리포트로 이동합니다.

### 1.2 🧪 검사별 프롬프트 (Test Prompts)
특정 검사 항목(예: 혈액검사, MRI, 내시경)을 하나만 콕 집어서 정밀하게 분석하고 싶을 때 사용하는 화면입니다.

*   **주요 기능**:
    *   **검사 선택**: 혈액, 소변, MRI, 초음파 등 분석하고 싶은 항목을 선택합니다.
    *   **환자 필터링**: 선택한 날짜에 해당 검사를 받은 기록이 있는 환자만 리스트에 표시됩니다.
    *   **개별 정밀 분석**: 선택한 검사 결과만을 AI에게 전달하여, 해당 분야 전문의 수준의 상세한 판독 소견을 생성합니다.

### 1.3 📝 종합소견 프롬프트 (Summary Prompt)
AI가 종합 소견을 작성할 때 사용하는 **'지시사항(프롬프트)'을 수정하고 테스트**하는 화면입니다.

*   **주요 기능**:
    *   **프롬프트 커스터마이징**: 좌측 에디터에서 AI에게 내리는 지시사항을 수정하고 저장할 수 있습니다. 병원마다 다른 소견 스타일(말투, 양식 등)을 적용할 때 사용합니다.
    *   **단일 환자 시뮬레이션**: 우측에서 특정 환자를 선택하여, 수정한 프롬프트가 잘 작동하는지 즉시 테스트해볼 수 있습니다. (새벽 배치를 기다리지 않고 바로 확인 가능)
    *   **상세보기 (모달)**: 생성된 결과를 팝업 형태로 미리 확인합니다.

### 1.4 📊 결과 리포트 (Result Report)
수검자(환자)에게 제공되는 최종 결과지 화면입니다.

*   **주요 기능**:
    *   **전문적인 디자인**: 병원의 품격에 맞는 깔끔한 레이아웃을 제공합니다.
    *   **이상 소견 강조**: 정상 범위를 벗어난 수치는 빨간색/주황색 등으로 표시되어 환자가 쉽게 알아볼 수 있습니다.
    *   **인쇄/PDF 저장**: A4 용지 규격에 맞춰 출력할 수 있습니다.

## 2. 데모 시연 순서 (Step-by-Step)

### 1️⃣ 종합 분석 프로세스 설명 (대시보드)
1.  상단 메뉴에서 **[종합소견 생성]** 클릭.
2.  **[배치 자동 생성]** 버튼을 가리키며 설명:
    *   "이 버튼이 바로 우리 솔루션의 핵심인 **'새벽 자동 분석'** 기능입니다."
    *   "매일 새벽 6시에 AI가 출근해서, 4단계(준비-수치-텍스트-종합) 과정을 거쳐 환자들의 소견을 미리 다 써놓습니다."
3.  리스트의 **'생성 완료'** 상태 확인.

### 2️⃣ 프롬프트 커스터마이징 시연
1.  상단 메뉴에서 **[종합소견 프롬프트]** 클릭.
2.  좌측 에디터를 보여주며 설명:
    *   "병원장님마다 원하시는 소견 스타일이 다르시죠? 여기서 AI에게 지시하는 내용을 마음대로 수정하고 저장할 수 있습니다."
3.  **[조회]** -> 환자 선택 -> **[AI 분석]** 클릭하여 변경된 스타일이 적용되는지 즉시 확인.

### 3️⃣ 결과지 출력 시연
1.  생성된 결과에서 **[상세보기 (모달)]** 또는 대시보드에서 **눈 아이콘(👁️)** 클릭.
2.  최종 결과지 확인 및 **[인쇄]** 버튼 클릭.

## 3. 자주 묻는 질문 (FAQ)

*   **Q: 새벽에 분석한다는 게 무슨 뜻인가요?**
    *   A: 진료 시간에 분석을 돌리면 시간이 걸릴 수 있으므로, 서버가 밤사이에 미리 환자 데이터를 분석해두는 기능입니다. 의료진은 출근 후 '완료된 결과'를 확인만 하면 됩니다.
*   **Q: 데이터가 없는 환자는 왜 안 보이나요?**
    *   A: 의료진의 편의를 위해, 해당 날짜에 검사 결과가 있는 환자만 자동으로 필터링해서 보여줍니다.
*   **Q: AI 분석 결과가 매번 똑같나요?**
    *   A: AI 모델의 특성상 문장의 표현은 조금씩 달라질 수 있지만, 핵심적인 의학적 판단(위험도, 진단명 등)은 일관되게 유지됩니다.
